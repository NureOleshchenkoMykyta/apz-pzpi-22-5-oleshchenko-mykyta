Короткий опис проекту: 

Проект: Emotional Analysis System 
	Це програмна система для моніторингу психоемоційного стану користувача. Вона складається з веб- та мобільного застосунку, що дозволяють фіксувати рівень стресу, додавати нотатки та переглядати історію емоційних результатів. Система підтримує авторизацію з розмежуванням ролей (користувач / адміністратор), інтернаціоналізацію інтерфейсу, а також реалізоване горизонтальне масштабування API через Kubernetes. Реалізовано резервне копіювання даних через JSON-файли, що забезпечує відновлення у разі збоїв, а також є можливість подальшої інтеграції з фізичними пристроями аналізу. 

 

Реалізовані функції: 

Мобільний застосунок з інтерфейсом для введення та перегляду результатів.  

Панель користувача з нотатками та історією результатів.  

Панель адміністратора з можливістю керувати акаунтами.  

REST API на Flask з авторизацією та розмежуванням ролей.  

Зберігання емоційних станів у базі PostgreSQL.  

Резервне копіювання та відновлення з .json-файлів (локальні дані).  

Горизонтальне масштабування API через Kubernetes (Minikube).  

Навантажувальне тестування системи за допомогою Locust.  

Симуляція IoT-пристрою (через текстове поле в мобільному застосунку).  

 

Не реалізовані функції: 

Автоматична передача біометричних даних від пристрою у базу.  

Інтеграція AI-моделей (прогнозування, аналіз поведінки).  

Інтеграція з медичними системами або іншими пристроями.  

Голосовий інтерфейс, гейміфікація, повноцінна медична аналітика.  

Пошук користувачів через ролі у адмін-панелі 

 

Висновок: 

У межах п’ятої лабораторної роботи було створено повноцінну демонстрацію системи аналізу психоемоційного стану користувачів. Під час презентації показано ключові функціональні можливості, реалізовані впродовж двох семестрів, а також проведено технічну демонстрацію масштабованості та стабільності системи. Проєкт охоплює як клієнтську, так і адміністративну частину, підтримує резервне копіювання та є готовим до подальшого розвитку.
